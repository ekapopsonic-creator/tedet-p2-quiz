# คลังข้อสอบ TEDET ป.2

เว็บแอพรวมข้อสอบจริง **TEDET ระดับประถมศึกษาปีที่ 2** วิชาคณิตศาสตร์และวิทยาศาสตร์ พร้อมเฉลย
เปิดหน้าโจทย์เป็นรูปภาพต้นฉบับ ฝึกทำแล้วตรวจคะแนนอัตโนมัติ และดูเฉลยละเอียดได้

## ชุดข้อสอบ
| ชุด | จำนวนข้อ | เฉลย |
|-----|---------|------|
| คณิตศาสตร์ 2567 | 30 | ✓ ตรวจคะแนนได้ |
| วิทยาศาสตร์ 2567 | 30 | ✓ ตรวจคะแนนได้ |
| คณิตศาสตร์ 2568 | 30 | ✓ ตรวจคะแนนได้ |
| วิทยาศาสตร์ 2568 | 30 | อ่านอย่างเดียว (รอไฟล์เฉลย ป.2) |
| คณิตศาสตร์ 2564 | 30 | อ่านอย่างเดียว (รอไฟล์เฉลย) |

## การใช้งาน
เปิด `index.html` ด้วยเบราว์เซอร์ หรือเข้าผ่านลิงก์ GitHub Pages ได้ทันที — ใช้งานออฟไลน์ได้ ข้อมูลคะแนนเก็บในเครื่องผู้ใช้

## ที่มา
ข้อสอบและเฉลยจาก TEDET (Thailand Educational Development and Evaluation Tests) — [tedet.or.th](https://www.tedet.or.th) เพื่อการฝึกฝนเท่านั้น
